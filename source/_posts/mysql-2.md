##### 创建数据库
`Mysql> Create Database [If Not Exists] test Charater set [gbk|utf8]`

##### 显示创建信息
`Mysql> Show Create Database test`

##### 修改数据库的字符集
`Mysql> Alter Database test character set = {gbk|utf8}`

##### 显示错误信息
`Mysql> Show warnings `

##### 删除数据库
`Mysql> Drop Database If Exists test `
